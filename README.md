# 🤖 AI Chatbot using NLTK (NLP Based)

## 📌 Project Overview

This project demonstrates the implementation of a basic AI chatbot using Natural Language Processing (NLP).  
The chatbot processes user input, tokenizes the text using NLTK, and generates responses based on keyword matching.

---

## 🚀 Technologies Used

- Python
- NLTK (Natural Language Toolkit)

## 🔍 Features

- Interactive chatbot running in terminal
- Tokenizes user input using NLTK
- Keyword-based intent recognition
- Continuous conversation loop
- Exit option to stop the chatbot

---

## 🛠 How It Works

1. The user enters a message.
2. The input text is converted to lowercase.
3. NLTK tokenizes the sentence into individual words.
4. The chatbot checks for specific keywords.
5. Based on detected keywords, it generates a response.
6. The loop continues until the user types `exit`.
